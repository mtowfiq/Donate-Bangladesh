What is DOM in JavaScript?

The Document Object Model (DOM) in JS is a very useful technique used to manipulate the HTML document using JS.

How can you select an element by it's ID in JavaScript?

We can select an element by it's ID using document.getElementById("id-name").

How do you change the inner text content of an HTML element using JavaScript?

First we get the element by it's id, and then use .innerText to set the inner text to whatever we like.
